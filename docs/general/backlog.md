# Backlog

## Spielkonzept

Wir programmieren ein Spiel. In diesem Spiel bewegt sich eine Person durch das Schulgebäude (Gänge und Räume). Durch Minigames werden Items gesammelt, die im Kampf mit Lehrer\*innen eingesetzt werden können. Durch gewonnene Lehrer\*innenkämpfe erhält man weitere Items für ein **Escape the School**.

## Gruppen

### 1. Szenengruppe
    * Gänge + Räume pixeln
    * Umsetzung der Karte / Umgebung
    * Umsetzung der Bewegung / Interaktion
    * Implementierung der Charaktere

### 2. Charaktere + Item Design
    * Charaktere + Items pixeln
    * Einbindung der Items

### 3. Minigames
    * Minigames pixeln + implementieren

### 4. Kampfgruppe
    * UI pixeln (für Kampf)
    * Implementierung Kampfsystem

### 5. Sounddesign
    * Musik + Soundeffekte
